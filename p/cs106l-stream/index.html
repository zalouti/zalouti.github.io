Stream Ⅰ

string stream

使用#include <sstream>

  • ostringstream代表output string stream
  • 创建一个stream时，默认位置在初始位置
  • 当尝试写入stream时，会覆盖原有内容
#include <sstream>
#include <iostream>

using namespace std;
int main()
{
    ostringstream oss("I am Icarus.");
    cout << oss.str() <<endl;
    oss << 16.9 << "AAAAAOnuce";
    cout << oss.str() <<endl;
    return 0;
}
  • 当然我们也可以将位置设置在末尾，即at end
    ostringstream oss("I am Icarus.",stringstream::ate);
  • istirngstream代表input string stream
  • istringstream将读取至下一个空格，并将读取的内容转化为对应的类型(double、int)
#include <sstream>
#include <iostream>

using namespace std;
int main()
{
    ostringstream oss("I am Icarus.");
    oss << 16.9 << "Onuce";
    istringstream iss(oss.str());
    double amount;
    string unit;

    iss >> amount >> unit;
    cout << "Now the water is "<< amount << unit;
    return 0;
}

我们可以制定ostringtream和istringstream所在的任意位置

    fpos pos = oss.tellp()+streamoff(3);
    oss.seekp(pos);

例一：string to integer

写一个函数，将输入的字符串转化为整数

int StringtoInteger (string str)
{
    istringstream iss(str);
    int value;
    iss >> value;
    return value;
}

但是这种解答不是完美的，我们并不知道iss >> value这一行是否成功了。

State bits

一共有四种表示stream的状态

  1. Good bit:准备好进行读写
  2. Fail bit:之前的操作失败了，所有未来操作被冻结
  3. EOF bit:end of file
  4. Bad bit:外部错误

Summary

stringstream

分为输出与输入ostringstream与istringstream

使用方法为：

istringstream iss("initial");
ostringstream oss("initial");

可以使用各种选项，例如：二进制、从末尾开始输入/出

istringstream iss("initial", stringstream::bin);
ostringstream oss("initial",stringstream::ate);

使用oss与iss从缓冲区中读取数据，将字符串变量转化为或者读取为对应的类型

oss << var1 << var2;
iss >> var1 >> var2;

我们可以手动调整stream在输入和输出时候的位置
