auto类型的使用

C++11支持自动类型，可以让编译器帮助你弄清这是什么类型

auto begin =studentMap.cbegin();
auto end =studentMAp.cend();

几乎所有的地方都可以使用auto，除了函数的形参不可以使用，例如下面的例子：

auto calculateeSum(const vector<string>& v){
    auto bettername =string{"Avery"};

如何使函数返回多个参数值？

对于CS106B来讲，函数返回值需要借助其他的reference parameters，例如下例

void findpricerange(int dist, int& min,int& max){
    min =static_cast<int>(dist*0.08+100);
    max =static_cast<int>(dist*0.36+750);
}
int main(){
    int dist =6452;
    int min ,max;
    findpricerange(age,max,min);
    cout <<"You can find price between:"
        <<min <<"and"<<max;
}

但是上面的做法并不能很明显地看出max与min是reference parameter，当我们返回一对值时会更加自然，例如下例：

pair <int,int>findpricerange(int dist){
    int min =static_cast<int>(dist*0.08+100);
    int max =static_cast<int>(dist*0.36+750);
    return make_pair(min,max);
}
int main(){
    int dist=6000;
    pair<int,int> p=findpricerange(dist);
    cout << "You can find prices between:"
        <<p.first <<"and"<<p.second;
}

结合auto我们可以将传回来的pair立马unpack，并直接使用min与max，只需将上面的

pair<int,int> p=findpricerange(dist);//pair不必类型相同

换为：

auto [min,max]=findpricerange(dist);

好多了！但是上面的程序还有缺点：

  1. 返回值时一对，能不能返回更多的值？tuple
  2. 返回值要求第一个是min第二个是max，但是程序使用者怎样知道呢？

所以更好的方式是使用struct，用法大致同C,略去不写

例如：

struct course{
string code;
TIme strattime;
Time endtime;
vector<string> instructors;
}
