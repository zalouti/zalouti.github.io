Sequence container

what is sequence container?

  • It provides the acsess to a sequence of elements
  • 元素是按照序列顺序储存的

Includes:

  • std::vector<T>
  • std::deque<T>
  • std::list<T>
  • std::array<T>
  • std::forward_list<T>

std::vector<T>

  • vector是一列任意元素类型的序列

  • T代表type，在使用时定义，这可以是任何的类型

    std::vector<int> vecInt;
std::vector<std::vector<string>> vecOfvec;

  • 可以完成所有vector可以完成的事情，同时可以很快速的完成push_front和pop_front
